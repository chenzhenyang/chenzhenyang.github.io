# Agentic Engineering Patterns 概述

**原文**: https://simonwillison.net/2026/Feb/23/agentic-engineering-patterns/  
**翻译**: LiterCrawler Skill

## 介绍

Agentic Engineering Patterns 是关于如何构建和管理 AI agent 系统的工程实践模式。

## 主要模式

### 1. 工具使用模式

Agent 通过调用外部工具来完成任务。

### 2. 多 Agent 协作

多个 Agent 协同工作，各自负责不同的任务。

### 3. 人类监督

在关键决策点引入人类监督，确保系统安全可靠。

### 4. 迭代优化

通过不断迭代和优化，提高 Agent 的性能和可靠性。

## 工程实践

- **模块化设计**: 将系统分解为独立的模块
- **可测试性**: 确保每个模块都可以独立测试
- **监控和日志**: 记录系统的运行状态
- **错误处理**: 建立完善的错误处理机制

## 结论

Agentic Engineering Patterns 为构建可靠的 AI agent 系统提供了指导。
